ゲームの核

散歩しながら話題を拾い、観客の反応が良い場所を見極めて熱量を上げる構成です。 ミスすると炎上雲やBAN板で体力が削られますが、ツッコミ波で切り返せます。

3つのステージ

商店街、川沿い、イベント会場を進みます。後半ほどコメント速度が上がり、 観客読みとジャンプ判断が重要になります。

リサーチ反映

長い配信歴、外歩き配信、時事トーク、打たれ強さ、場の空気を読む巧さを システムへ翻訳しました。詳しくは README に整理しています。
