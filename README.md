# 野田草履の冒険島

公開情報をもとにしたファンゲームのプロトタイプです。実在の人物の写真や音声は使わず、配信スタイルと文脈をゲームシステムへ翻訳しています。

## 今回のリサーチ要点

以下の公開情報を軸に、キャラクター性をそのまま消費するのではなく、ゲーム上の役割へ変換しました。

1. ツイキャスの公開プロフィールでは、`ライブストリーマー。ニコ生やYoutubeやってます。` とされ、公開アーカイブにも `秋を探しにいく` `金バエさんに熊手を届ける` `うちの近所のビルが大火災` など、外を歩きながら出来事に反応する配信タイトルが並んでいる。
2. 2024年12月時点のツイキャス公開ページでは、サポーター数 7万人規模、アーカイブ 6.4k 以上が見え、長い継続活動と視聴導線の強さが確認できる。
3. 2017年のニコニコニュース記事では、野田草履さん自身が `10年間引きこもって` いた過去を語っており、配信上の剥き出しさや打たれ強さの背景として参照できる。
4. 同じく2017年のニコニコニュース記事では、横山緑さんが `ちゃんと人を選ぶ` と評しており、観客の反応が良い相手や場面を読む能力が強みとして見える。
5. ニコラジ記事では、ニコ生と他プラットフォームを比較しながら継続配信の収益や現実感を語っており、単なる勢いだけでなく `配信という営みを続ける人` としての顔がある。

## ゲーム設計への落とし込み

- `外配信` は横スクロールの移動アクションに変換
- `コメント処理` はコメント玉回収でスコアと熱量を上げる仕組みに変換
- `話題の拾い方` はネタ札の高得点アイテムとして変換
- `打たれ強さ` は残機制と短時間の無敵フレームに変換
- `観客読み` は終盤ほど判断が必要な配置と、ツッコミ波の切り返し能力に変換
- `炎上や危うさ` は炎上雲とBAN板として抽象化

## ファイル

- `index.html` UI と画面構成
- `styles.css` レトロ配信会場風の見た目
- `game.js` ゲーム本体

## 起動

`index.html` をブラウザで開くだけで遊べます。

## 補足

実在人物ベースの題材なので、今後さらに作り込むなら以下の方針が安全です。

- 私生活や未確認情報ではなく、公開配信上の文脈だけを扱う
- 暴力や事件そのものを面白がる内容には寄せない
- 本人写真や第三者の権利物は使わない
- `野田草履風の配信アクション` として抽象度を維持する
